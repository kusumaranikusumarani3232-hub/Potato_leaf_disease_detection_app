# 🥔 Potato Leaf Disease Detection Using CNN

A Deep Learning based Potato Leaf Disease Detection system built using TensorFlow, Keras, and Streamlit.

This application predicts potato leaf diseases from uploaded leaf images using a Convolutional Neural Network (CNN) model.

---

# 🚀 Features

* Potato leaf disease prediction
* CNN based image classification
* Upload image and get prediction instantly
* Confidence score display
* Streamlit web application
* Deep Learning image processing pipeline

---

# 🛠 Technologies Used

* Python
* TensorFlow
* Keras
* Streamlit
* NumPy
* Matplotlib
* Pillow

---

# 🧠 Deep Learning Workflow

Image Upload
↓
Image Preprocessing
↓
CNN Model Prediction
↓
Disease Classification

---

# 📊 Classes Predicted

* Potato Early Blight
* Potato Late Blight
* Healthy Potato Leaf
